Getting Started

What you need:

  • Basic knowledge of HTML
  • A code editor, like VS Code or Atom
  • Basic understanding of design

To help you get started with CSS, we've provided some basic HTML boilerplate below.

<!DOCTYPE HTML> 
<html lang="en">
<head>
<meta charset="UTF-8"/>
<meta name="viewport" content="width=device-width, initial-scale=1.0"/>
<title> Hello World! </title>
</head>
<body>
<h1>Hello world</h1>
<p>this is an introductory webpage to teach you the basic code of an html document!</p>

Once you have a basic HTML document you want to style, you need to create and link your CSS document. To link your document use the following code and place in in the head of your HTML document.

 <link rel="stylesheet" href="styles.css">

Now you're ready to start coding with css!

in css, you always start with a selector. Selectors are most often associated with

  • Classes
  • IDs
  • Elements

You can see a full list of CSS selectors on W3 schools.com Begin your css document using the basic syntax below.

 
        p{
        color:red;  
        }

If your document is set up properly, any text in a <p> element should be red.

Coloring

In CSS you can set the color of various elements on your webpage. There are multiple ways to set the color value:

  • RGB
  • Hex
  • HSl
  • RGBA
  • Color Name

On our getting started page, we chaged the text color using the color name red.

Lets use the RGB property to change the background-color of our h1 element to blue:

 
 
         h1{
         background-color:rgb(0 , 0, 255);
         }

When using the RGB color property, the first number represents the red value, the second number represents the blue value and the third value represents the blue value. You can put any number between 0 and 255.

Now that we have a blue background on on our h1 element, the red text is hard to see.Lets use a hex code to change the text color to white.


         h1{
         background-color:rgb(0, 0, 100);
         color:#ffffff;
         }

Hex codes work similaly to RGB. The first two characters represent the red value, the second two characters represent the green value and the last two characters represent the blue value. The values are comprised of the numbers 1-9 and letters a-f. the higher the letter or number, the higher the value.

now we want to make our P elements match our h1 element. We can start by changing the background color to blue using the hsl method. 


         p{
         background-color:hsl(240, 100%, 50%)
         }

hsl stands for hue, saturation, lightness. if you've done design work before, you have probably used the hsl color wheel in the past. The hue value goes from 0 to 360, the other two saturation and lightness are percentages. the higher the percentage the higher the saturation or lightness

Now we need to chabge the text color to white. We can do this using the RGBA method. Rgba stands for red, green, blue, alpha. the alpha number allows us to change the opacity of the color using a percentage value. 


         p{
         color:hsl(240 ,100% ,50% );
         background-color:rgba(255 ,255 ,255, 50%);
         }

Sizing

CSS allows you to resize many of your elements including:

  • Text
  • Images
  • Text Elements
  • Container Elements
  • Videos

Text can be resized using various measurements. The first method is to enter the size of the text as xx-small, x-small, small, medium, large, x-large or xx-large. Lets set our h1 element to x-large

          
         h1{
         background-color:rgb(0, 0, 100);
         color:#ffffff;
         font-size:x-large;
         }

You can also resize the text using units of measure:

  • px
  • em
  • rem

Each of these measurements is relative to another element, meaning the size of the text will change as the size of the relative element changes.

You can also re-size images, Text-Elements and container elements by setting their width and height. When working with a website that needs to be compatible with both computers and mobile devices, you should use the percent method, or the viewport height (vh) and viewport width (vw), when setting the size of an element. lets go ahead and set the height of our body to 100vh.

         
         body{
         height:100vh;
         }

Now our height will always take up 100% of the viewport. We should also set the width to 100vw.

            
         body{
         height:100vh;
         width:100vw;
         }

Once you've set your width and height, you can begin setting margins and padding. when setting these elements, you may end up adding to your width and height, so you may need to play around with your sizing properties to ensure your page appears how it's supposed to.

Display and Grids

The CSS Display Property allows you to re-arrange the structure of your webpage. We will go over the basic display properties.

  • Block

    • The block display property is usually the default given by the html code to an element. This property allows items to display below each other, adding line breaks between elements.

    
         p{
         display:block;
         }

  • Inline-block

    • The Inline-block display property allows you to display items side by side. This property does not use line breaks.

    
         h1{
         display:inline-block;
         }

  • Flex

    • The Flex display property is the most diverse display property. It allows items to display in grid like rows and columns.To customize the way your flex item is displaying, you will need to use the flex-direction property. The flex direction property can be set to row, which displays items side by side, or column which displays items one below the other. Finally, when working with flex boxes, you will need to use the Justify-content property to achieve the desired alignment of your flex items. 

    
          body{
          display:flex;
          flex-direction:row;
          justify-content:center;
          }

  • Grid

    • The Grid property allows you to display your items in a grid format. First you will need to set the display property to grid.

    
         body{
         display:grid;
         }

    Next you will need to set the column gaps and the row gaps.

    
         body{
         display:grid;
         row-gap:30px;
         column-gap:30px;
         }

    now you should have an evenly spaced grid to display your content in!
